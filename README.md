## ğŸ“– Usage

### Basic Workflow

1. **Write TSX Code** - Use the editor on the left to write TSX code
2. **Generate Video** - Click the "Generate Video" button
3. **Watch Preview** - See your video render in real-time on the right
4. **Download** - Click the download to save your MP4

### Example TSX Code

Generate TSX using claude by following steps

1. Open [Claude AI](https://claude.ai/) and add new project in Projects section.
2. Edit the instructions section by copying the content from the instructions file.> ğŸ’¡ **Tip:** The instructions file can be found in the repository root as `Instructions.txt`
3. After Saving the instructions, write a prompt like below. This is one example. For more prompts please do some exploration

   ```
   Create a 5-second countdown from 5 to 1 with neon glow effect

   ```

4. Claude AI generates TSX code, copy the code and paste in the TSX code Editor and click Generate button.
5. After few moments, the video will be rendered on the right side of the UI
6. Select options on the video to download.

## ğŸ”§ Troubleshooting

### Video not rendering?

1. Check that the backend API is running on port 3000
2. Verify your TSX code
3. Check browser console for errors
4. Ensure all Remotion dependencies are valid

### Port already in use?

Change the port in `vite.config.ts`:

```ts
server: {
  port: xxxx, // Change this
}
```

### Build errors?

```bash
# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install
```
